# Érosion des Âmes - Alpha

Projet de refonte complète du jeu "Érosion des Âmes".

## Technologies Utilisées

### Frontend
- **Framework**: Vite + React
- **Styling**: TailwindCSS v3
- **Routing**: React Router

### Backend
- **Runtime**: Node.js 22.17.0
- **Framework**: Express
- **ORM**: Sequelize

### Base de Données
- **SGBD**: MySQL

### Hébergement
- **Provider**: O2Switch

### Versioning
- **Git**: GitHub

## Structure du Projet

```
erosion-des-ames-alpha/
├── frontend/          # Application React
├── backend/           # API Express
├── docs/              # Documentation technique
└── README.md          # Ce fichier
```

## Installation

### Prérequis
- Node.js 22.17.0
- MySQL
- Git

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
